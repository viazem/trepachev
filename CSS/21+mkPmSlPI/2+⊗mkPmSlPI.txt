[!ЗАМЕЧАНИЕ] Тега <b> нет в коде.

Предположим, что имеется в виду, что тег <i> это тег <b>.

<p>
	This is some <b>text</b> in paragraph.
</p>

p {
	color: blue;
}

b {
	color: red;
}

Изучив код, скажите какой цвет будет иметь текст в теге b:

Ответ:  красный цвет.
